# lottoi-googleurl-verification-log
로또아이는 각 회차마다 회원이 생성한 번호를 추첨 이전 시점에 외부 문서 형태로 기록합니다.  해당 외부 문서는 회차별로 고유한 URL을 가지며, 생성 시점과 이후 변경 이력이 자동으로 남습니다.  이 저장소는 그러한 구글 공개 기록 URL이 생성되었다는 사실을 회차별·시간 순서대로 다시 기록하는 공개 로그 공간입니다.  이를 통해 URL의 생성 시점과 기록 시점을 함께 비교할 수 있으며, 결과 기록의 생성 시점과 변경 이력을 공개함으로써 사후 변경 여부를 누구나 외부에서 확인·검토할 수 있는 공개 검증 구조를 제공합니다.
