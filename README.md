# lottoi-googleurl-verification-log

로또아이는 각 회차마다 회원이 생성한 번호를  
**추첨 이전 시점에 외부 문서 형태로 기록**합니다.  

해당 외부 문서는 회차별로 **고유한 URL**을 가지며,  
**생성 시점과 이후 변경 이력**이 자동으로 남습니다.

---

본 저장소는 위와 같은 **구글 공개 기록 URL이  
추첨 이전에 생성되었다는 사실**을  
**회차별·시간 순서대로 기록**하는 **공개 로그 공간**입니다.

---

이를 통해 **URL의 생성 시점**과  
**본 저장소의 기록 시점**을 함께 비교할 수 있으며,  

결과 기록의 **생성 시점과 변경 이력**을 공개함으로써  
**사후 변경 여부**를  
누구나 외부에서 **확인·검토**할 수 있는  
**공개 검증 구조**를 제공합니다.
