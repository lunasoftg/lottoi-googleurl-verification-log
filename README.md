# lottoi-googleurl-verification-log

로또아이는 각 회차마다 회원이 생성한 번호를  
**추첨 이전 시점**에 **외부 문서 형태**로 기록합니다.

이 외부 문서는 회차별로 **고유한 URL**을 가지며,  
문서가 **언제 처음 생성되었는지**,  
그리고 이후에 **어떤 변경이 있었는지**에 대한  
시간 기록이 자동으로 남습니다.

---

이 저장소는 위와 같은 **구글 공개 기록 URL이  
추첨 이전에 이미 생성되어 존재했음을**  
회차별·시간 순서대로 남기기 위한  
**공개 로그 공간(verification log)**입니다.

즉, 구글 문서 자체를 대신 검증하거나 보증하는 곳이 아니라,  
해당 URL이 **언제부터 존재했는지**를  
외부에서 다시 한 번 확인할 수 있도록  
시간 기준의 기록을 공개적으로 남기는 역할을 합니다.

---

이를 통해 구글 공개 기록 URL의 **생성 시점**과  
본 저장소에 기록된 **로그 시점**을 함께 비교할 수 있으며,  

결과 기록의 생성 시점과 변경 이력이  
사후에 바뀌었는지 여부를  
누구나 외부에서 **확인·검토**할 수 있는  
**공개 검증 구조**를 제공합니다.


---
